# Personal Dashboard

This is a personal dashboard designed for individual use. I made this project to make myself productive; it tracks everything in my daily routine so I can see my progress.

## Features

### Dashboard
- Clock
- Weather
- Code snippets
- Calendar overview

### Productivity
- DeepWorks print
- Most important task tracker
- Output tracker

### Journals and Notes
- Personal journal
- AI-assisted journal
- Notes
- AI-powered notes with an assistant

### Side Links
Quick access to frequently used websites and tools.

### News
- Personalized news feed

## Overview
The dashboard centralizes personal productivity, organization, and daily tracking in a single interface, helping monitor progress and stay productive.
